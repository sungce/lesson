02

개념설명

⏱ 15분▾

입력 장치 (Input Device)

사용자나 외부 환경으로부터 데이터를 컴퓨터에 입력하는 장치입니다. 사람의 명령을 컴퓨터가 처리할 수 있는 디지털 신호로 변환합니다.

  • 키보드 — 문자·숫자 입력
  • 마우스 — 위치·클릭 입력
  • 마이크 — 소리 입력
  • 웹캠 — 영상 입력
  • 스캐너 — 이미지 입력
  • 터치스크린(입력 부분) — 터치 위치 입력

출력 장치 (Output Device)

컴퓨터에서 처리된 결과를 사람이 인식할 수 있는 형태(시각·청각·촉각 등)로 출력하는 장치입니다.

  • 모니터 — 영상 출력
  • 스피커 / 이어폰 — 소리 출력
  • 프린터 — 인쇄물 출력
  • 프로젝터 — 대형 화면 출력
  • 진동 모터 — 촉각 출력 (스마트폰)
⚠️
혼동 주의 — 터치스크린: 터치스크린은 화면을 보여주는 출력과 손가락 터치를 감지하는 입력을 동시에 수행하는 복합 장치입니다. 하나로 단정 짓지 마세요!
