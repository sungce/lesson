02

개념설명

⏱ 15분▾

CPU — 중앙처리장치

컴퓨터의 "두뇌"입니다. 모든 연산과 제어를 담당합니다. CPU 성능은 클럭 속도(GHz)와 코어 수로 결정됩니다.

  • ALU (산술논리연산장치): 덧셈·뺄셈·AND·OR 등 모든 연산 수행
  • CU (제어장치): 명령어 해석, 전체 흐름 지휘
  • 레지스터: CPU 내부 초고속 임시 기억 공간
  • 클럭: 초당 처리 사이클 수 (3.5GHz = 35억 회/초)

메모리 계층 구조

메모리는 속도가 빠를수록 용량이 작고 비쌉니다. 아래로 내려갈수록 느리고 용량이 큽니다.

  • 레지스터: CPU 내부, 가장 빠름, 수십 byte
  • 캐시 메모리: CPU↔RAM 사이 완충, MB 단위
  • RAM: 임시 기억장치, 전원 끄면 삭제, GB 단위
  • SSD/HDD: 보조기억장치, 영구 저장, TB 단위
⚠️
혼동 주의: 스마트폰 사양의 "메모리 8GB"는 RAM이고, "저장 용량 256GB"는 보조기억장치(낸드플래시)입니다. 둘은 전혀 다릅니다!
