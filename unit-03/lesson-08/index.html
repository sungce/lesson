02

개념설명

⏱ 15분▾

리스트(List)란?

여러 개의 데이터를 순차적으로 저장하는 공간입니다. 변수가 하나의 값이라면 리스트는 여러 값을 묶어 관리합니다.

  • 인덱스: 각 항목의 번호 (1번부터)
  • 추가: 항목 끝에 새 값 추가
  • 삭제: 특정 번호 항목 제거
  • 수정: 특정 번호 값 변경
