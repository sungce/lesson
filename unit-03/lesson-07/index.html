02

개념설명

⏱ 15분▾

중첩 제어 구조란?

반복문이나 조건문 안에 또 다른 반복문·조건문이 있는 구조입니다. 외부 반복 1회마다 내부 반복이 전체 실행됩니다.

반복 속 반복
🔁 6번 반복 (외부)
🔁 4번 반복 (내부)
📐 이동·그리기
↪️ 회전
→ 6×4 = 24번 실행
반복 속 조건
🔁 계속 반복하기
🔀 만약 점수≥100 이라면
⛔ 모든 코드 멈추기
🟰 점수 ← 점수+1
→ 100점에서 정지
⚠️ 외부 반복 1회 = 내부 반복 전체 실행. 총 횟수 = 외부×내부
📖 교과서 연계: pp.124~131 / 문양 그리기 + 안전 지킴이 프로그램
