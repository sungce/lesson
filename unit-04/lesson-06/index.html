02

개념설명

⏱ 15분▾

AI 개발 단계별 윤리적 고려 사항

🔍 문제 정의 단계
사람에게 이로운 프로그램인지 확인 → 개발 여부 판단
예: 얼굴형 추천 AI → 특정 외모 차별 가능성 고려
📥 데이터 수집 단계
① 개인정보 보호: 동의서 받기 + 비식별화 처리
② 저작권: 저작물 사용 허락 받기
③ 비식별화: 이름→가명, 개인식별 정보 제거·변형
🤖 모델 학습 단계
데이터 편향: 한쪽으로 치우친 데이터 → 불공정 결과
예: 특정 성별 데이터만 학습 → 차별적 예측 결과
✅ 활용 단계
AI 결과가 개인·사회에 미치는 영향 지속 모니터링
윤리 기준 준수 여부 점검
⚠️ 핵심: AI는 학습 데이터의 편향을 그대로 학습합니다. 다양하고 균형 잡힌 데이터 수집이 공정한 AI의 첫걸음입니다.
📖 교과서 연계: pp.199~203
