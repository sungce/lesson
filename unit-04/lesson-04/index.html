02

개념설명

⏱ 20분▾

소리 분류 모델

소리 데이터의 특성을 분석하여 특정 소리가 어떤 범주에 해당하는지 예측하는 모델입니다.

  • 음성 인식 키오스크: 메뉴 소리 인식
  • AI 스피커: 음성 명령 인식
  • 보안 시스템: 특정 소리 감지
  • 의료: 숨소리·기침으로 상태 진단

웃음 계수 시스템 문제 해결 과정

문제 정의: 웃음·화남·울음·생활 소음을 학습 → 웃을 때마다 횟수 카운트
데이터 수집: 4가지 소리 데이터 녹음 → 훈련/테스트 분류
모델 생성: 티처블 머신 소리 분류 모델 학습 · 평가
프로그램 적용: 엔트리에서 모델 연동 → 웃음 감지 시 카운트 증가
📖 교과서 연계: pp.186~191 / 소리 분류 + 엔트리 웃음 카운터
