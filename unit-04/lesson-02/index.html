02

개념설명

⏱ 15분▾

AI에서 데이터의 역할

인간이 지식과 경험으로 대상을 인식하듯, AI는 데이터 학습을 통해 대상을 인식·예측합니다. 더 많은 데이터로 반복 학습할수록 AI 성능이 개선됩니다.

  • 이미지 데이터: 사물 인식 (로봇 청소기)
  • 소리 데이터: 음성 인식 (AI 스피커)
  • 텍스트 데이터: 의미 파악 (온라인 쇼핑)
  • 움직임 데이터: 자세 교정 (AI 카메라)

데이터 수집 방법

📷 직접 수집
카메라로 이미지, 마이크로 소리를 직접 촬영·녹음하여 수집
🌐 웹·DB 수집
검색 엔진이나 데이터 전문 웹사이트에서 미리 생성된 데이터 수집

레이블링과 데이터 분류

레이블(Label): 데이터에 부여하는 정답 태그 (예: 고양이 이미지 → '고양이')
레이블링: 데이터에 레이블을 부여하는 과정
훈련 데이터: AI 모델 학습에 사용 (~80%)
테스트 데이터: 학습 결과 평가에 사용 (~20%)
예시: 스팸 메일 — '일반 메일' / '스팸 메일' 레이블 → 학습
⚠️ 오개념: 데이터 양이 많다고 항상 좋은 것이 아닙니다. 레이블링이 잘못된 데이터는 오히려 AI 성능을 낮춥니다.
