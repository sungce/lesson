02

개념설명

⏱ 15분 ▾
알고리즘(Algorithm)이란?

어떤 문제를 해결하기 위한 절차나 방법을 단계적으로 나타낸 것입니다. 좋은 알고리즘은 5가지 특성을 갖습니다: 입력, 출력, 명확성, 유한성, 유효성.

입력 → 처리 → 출력 흐름
순서도(Flowchart) 기호

알고리즘을 시각적으로 표현하는 도구입니다. 각 도형은 의미가 정해져 있습니다.

⚠️
자주 혼동하는 개념: "알고리즘"과 "프로그램"은 다릅니다. 알고리즘은 문제 해결 절차를 설명한 계획이고, 프로그램은 그 알고리즘을 컴퓨터가 이해할 수 있는 언어로 구현한 것입니다.

03

인터랙션

⏱ 15분 ▾
💡 이 활동을 통해 알고리즘의 순서와 조건 판단이 어떻게 동작하는지 직접 확인할 수 있습니다.
퀴즈 코딩 실습
다음 중 알고리즘의 특성으로 올바르지 않은 것은?
① 유한성: 알고리즘은 반드시 유한한 단계 후에 종료되어야 한다.
② 명확성: 각 단계의 지시가 명확하고 모호하지 않아야 한다.
③ 무한성: 알고리즘은 무한히 반복되어야 더 정확한 결과를 낸다.
④ 유효성: 각 단계는 실제로 수행 가능한 연산이어야 한다.
Python · starter 코드 
# 최댓값 찾기 알고리즘
# 빈칸을 채워 완성해보세요!

def find_max(numbers):
    # 첫 번째 값을 최댓값으로 초기화
    max_val = numbers[0]
    
    for num in numbers:
        # 조건: 현재 값이 max_val보다 크면?
        if num _____ max_val:   # 빈칸 채우기
            max_val = num
    
    return max_val

# 테스트
data = [3, 7, 1, 9, 4]
print("최댓값:", find_max(data))
>>> 최댓값: 9
💬 예상 출력: 최댓값: 9
