02

개념설명

⏱ 15분▾

문자의 디지털 표현 — 아스키(ASCII)

컴퓨터는 문자를 숫자(이진수)로 저장합니다. 아스키 코드는 영문자·숫자·특수문자를 7비트(128가지)로 약속한 표준입니다.

  • A = 65₁₀ = 01000001₂
  • a = 97₁₀ = 01100001₂
  • 0 = 48₁₀ = 00110000₂
  • 유니코드: 한글·한자·이모지 포함 전 세계 문자

이미지의 디지털 표현 — 픽셀

디지털 이미지는 수많은 작은 점인 픽셀(Pixel)로 구성됩니다. 각 픽셀은 RGB 세 가지 색의 밝기(0~255)로 색상을 표현합니다.

  • 해상도: 가로×세로 픽셀 수 (예: 1920×1080)
  • RGB: 빨강(R)·초록(G)·파랑(B) 각 0~255
  • 24비트 색상: 픽셀 하나 = 3바이트
⚠️
혼동 주의: 해상도는 화면 크기가 아닙니다. 같은 크기 화면이라도 픽셀 수가 많으면(고해상도) 더 선명하게 보입니다.
