02

개념설명

⏱ 15분▾

비트(bit)와 바이트(byte)

컴퓨터가 처리하는 가장 작은 단위는 비트(bit)로, 0 또는 1 중 하나의 값만 가집니다.

  • 1 bit = 0 또는 1 중 하나
  • 1 byte = 8 bit
  • 1 KB = 1,024 byte
  • 1 MB = 1,024 KB

10진수 → 2진수 변환 (나눗셈법)

예시: 13을 2진수로 변환
13 ÷ 2 = 6 … 나머지 1 ← 최하위 비트
 6 ÷ 2 = 3 … 나머지 0
 3 ÷ 2 = 1 … 나머지 1
 1 ÷ 2 = 0 … 나머지 1 ← 최상위 비트

아래→위로 읽으면: 1101₂

2진수 → 10진수 (자릿값법)

예시: 1101₂ → 10진수
1×2³ + 1×2² + 0×2¹ + 1×2⁰ = 8 + 4 + 0 + 1 = 13
