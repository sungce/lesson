02

개념설명

⏱ 15분▾

데이터 유형 3가지

  • 정형 데이터: 표(행·열) 구조. 엑셀, DB 테이블
  • 비정형 데이터: 구조 없음. 이미지·텍스트·영상·SNS
  • 반정형 데이터: 부분 구조. JSON, XML (태그 기반)

다음 중 정형 데이터(Structured Data)의 특징으로 옳은 것은?
①이미지나 영상처럼 구조가 없다.
②행과 열로 구성된 표 형태로 저장된다.
③SNS 게시글처럼 자유로운 형식이다.
④JSON 형식처럼 태그와 값으로 구성된다.
