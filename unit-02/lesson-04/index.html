02

개념설명

⏱ 15분▾

데이터 유형

데이터는 구조에 따라 크게 두 가지로 나뉩니다.

  • 정형 데이터: 표(행·열) 형태. 엑셀, DB 테이블
  • 비정형 데이터: 구조 없음. 텍스트·이미지·영상·SNS
  • 반정형 데이터: JSON, XML (중간 형태)

데이터 수집 방법 비교

수집 방법 특징 예시
설문조사사람에게 직접 질문구글폼, 종이 설문
센서 수집자동 측정·기록온도계, 미세먼지 센서
공공데이터정부·기관 공개 자료data.go.kr, 통계청
웹 스크래핑웹사이트 자동 수집뉴스·쇼핑몰 가격
🌐 공공데이터 활용: data.go.kr에서 미세먼지·날씨·인구 등 다양한 공공데이터를 무료로 다운로드할 수 있습니다.

03

인터랙션 — 데이터 유형 분류 퀴즈

⏱ 15분▾
💡 제시된 데이터 사례를 정형·비정형·반정형으로 분류하고, 적절한 수집 방법을 매칭해보세요!
▸ 아래 데이터의 유형을 선택하세요
📊
학생 성적 엑셀 파일
이름·점수·등급이 행과 열로 정리된 표
📸
인스타그램 사진
사용자가 업로드한 이미지 파일
📡
날씨 API 응답 (JSON)
{"temp":22, "humidity":65}
📰
뉴스 기사 텍스트
정해진 구조 없이 서술된 글
다음 중 정형 데이터(Structured Data)의 특징으로 옳은 것은?
①이미지나 영상처럼 구조가 없다.
②행과 열로 구성된 표 형태로 저장된다.
③SNS 게시글처럼 자유로운 형식이다.
④JSON 형식처럼 태그와 값으로 구성된다.
